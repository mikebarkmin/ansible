# Prerequesit

Only a minial installation is needed for this script to work.

- ansible
- neovim
- zsh
- flatpak-builder
- git
- python3
- python3-psutil
- stow
- tmux
- fzf
- wl-clipboard

Create an inventory file at `$HOME/.config/ansible/inventory`

```
[all]
silberkarpfen
```

or

```
[all]
silberfisch
```
